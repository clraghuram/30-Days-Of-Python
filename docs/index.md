## What is this?

A hands-on curriculum. Each day includes:

- Clear explanations
- Real runnable code examples
- Exercises to practice

Work at your own pace. Everything is free and open source.

## How to use

1. Go through days in order (each builds on the last)
2. Type and run the code yourself
3. Try the exercises
4. Come back to previous days as reference

## Course roadmap

**Days 1–10** — Core foundations  
Variables, operators, strings, lists, tuples, sets, dicts, conditionals, loops

**Days 11–20** — Everyday tools  
Functions, modules, comprehensions, errors, regex, files, packages

**Days 21–30** — Real world  
Classes, web scraping, virtualenvs, Pandas, web, MongoDB, REST APIs

## Prerequisites

- A computer
- Python 3.8+
- Any text editor (VS Code recommended)
