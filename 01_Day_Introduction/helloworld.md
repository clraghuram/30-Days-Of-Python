# Day 1 — Hello World

Welcome to **30 Days of Python**!

Today we write our very first Python program and get comfortable with the basics.

## Your first program

```python
print("Hello, World!")
```

**Output:**
```
Hello, World!
```

## Try it yourself

1. Open a terminal
2. Type `python3` (or `python`) to start the interactive shell
3. Type the line above and press Enter

Or save it as a file:

```bash
# Create the file
echo 'print("Hello, World!")' > hello.py

# Run it
python3 hello.py
```

## What just happened?

- `print()` is a built-in function that outputs text to the screen.
- Everything inside the quotes is a **string** (text data).
